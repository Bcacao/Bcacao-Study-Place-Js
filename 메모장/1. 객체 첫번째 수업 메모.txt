- 객체 생성 방법 
프로퍼티의 키값을 문자열로 써도 된다
문자열로 쓸때는 ['']을 사용 하면 된다

- 객체를 부르는 방법
일반 변수처럼 부르면 된다

- 객체의 특정 프로퍼티를 부르는 방법
점을 써서 부르면 된다
프로프티를 번수처럼 저장하여 [''] 형태로 부를 수 있다

- 객체 프로퍼티 수정 방법
- 객체 프로퍼티 추가 방법
추가랑 삭제 방법은 비슷하다 
객체의 특정 프로퍼티를 부르는 방법을 통해 변수 수정 추가 방법과 같게 만들면 된다

- 객체 프로퍼티 삭제 방법
delete를 사용하면 된다

- 객체의 프로퍼티를가 있는지 확인 방법
in을 사용하여 확인 가능하다
in을 사용할 경우 프로퍼티의 키값을 문자열로 물어봐야한다
반복문을 사용하여 추출할 수도 있다
